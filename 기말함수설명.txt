함수설명

mouse_move
>> 마우스 움직이는 거

mouse_press
>> 마우스 클릭 할 때

draw_neko
>> NEKO 배열에 있는 숫자를 기준으로 그림을 삭제

check_neko
>> 퍼즐 요소에 맞는 배치 확인

sweep_neko
>> 퍼즐 요소에 맞는 블럭을 빈 블럭으로 변경

drop_neko
>> 블랙 아래칸을 검사하여 비어있는 경우 블럭을 아래로 이동

over_neko
>> 게임 종료

set_neko
>>첫 줄에 블럭을 생성함

draw_txt
>> 일반적인 텍스트를 배치

game_main
>> stage로 나누어져 있음
0. 로고 >> 1로 넘어감, 조건 없음
1. 타이틀 화면, 대기화면 >> 2로 넘어감, 조건은 난이도 선택 완료 시
2. 블럭 낙하(drop,draw) >> 3번으로 넘어감, 블럭 낙하가 발생하지 않음
3. 블럭 확인(check, draw) >> 4번으로 넘어감, 조건 없음 
4. 블럭 삭제
>> 점수가 있으면 2번으로,
점수가 없으면(조건을 만족하지 않으면) 5번으로 넘어감
조건을 만족하면 6번으로 넘어감
5. 마우스 입력 대기 >> 2번으로 넘어감, 클릭하면 
6. 게임 오버 >> 0번으로 넘어감, 타이머가 50이 되면
